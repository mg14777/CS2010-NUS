# CS2010-NUS

Solutions for Problem Sets of Data Structures and Algorithms II at National University of Singapore for Semester 2 AY2016/17

The themes for different problem sets are given below

<b>Problem Set 1</b>: Heaps / Priority Queues<br>
<b>Problem Set 2</b>: Self Balancing Trees (AVL trees)<br>
<b>Problem Set 3</b>: General graph search (BFS/DFS)<br>
<b>Problem Set 4</b>: Minimum Spanning Trees (Kruskal/Prim's)<br>
<b>Problem Set 5</b>: Shortest Path Problems<br>
<b>Problem Set 6</b>: Travelling Salesman Problem (Dynamic Programming)<br>
